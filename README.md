# 20-Minute Dumbbell Routine

A clean, mobile-friendly workout reference card hosted on GitHub Pages. Built for a beginner lifter with a set of adjustable dumbbells, focused on building overall strength and muscle.

## Live Site

[View the routine](https://your-username.github.io/your-repo-name/)

> Replace the link above with your actual GitHub Pages URL once deployed.

## What's Inside

A single `index.html` file — no frameworks, no dependencies beyond a Google Fonts import. The page includes:

- **5 full-body exercises** with muscle group callouts
- **Plain-English descriptions** for each movement
- **YouTube links** for beginner form tutorials
- **Key tips** on progressive overload and rest
- Format details: 40 sec on / 20 sec off, 3 rounds, ~20 minutes

## The Exercises

1. Goblet Squat — Legs & Glutes
2. Romanian Deadlift — Hamstrings & Lower Back
3. Bent-Over Row — Back & Biceps
4. Shoulder Press — Shoulders & Triceps
5. Floor Chest Press — Chest & Triceps

## Deploying to GitHub Pages

1. Create a new GitHub repository
2. Add `workout-routine.html` to the repo and rename it `index.html`
3. Go to **Settings → Pages**
4. Under **Source**, select `Deploy from a branch`
5. Choose `main` branch and `/ (root)`, then click **Save**
6. Your site will be live at `https://your-username.github.io/your-repo-name/` within a minute or two

## Customization

All styles are in a single `<style>` block inside `index.html`. Key values to change:

| What | Where |
|---|---|
| Accent color (gold) | `#c8a96e` — find & replace all instances |
| Dark background | `#0d0d0d` |
| Cream background | `#ede8e1` |
| Footer tagline | Bottom of `<body>`, `footer-badge` span |
| Exercise content | Each `.exercise` block in the HTML |
